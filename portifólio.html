Projetos Desenvolvidos

Estes são dois trabalhos acadêmicos que desenvolvi durante o curso de Análise e Desenvolvimento de Sistemas. Cada um deles me ajudou a fortalecer diferentes habilidades da área.

🧠 Projeto 1: Aplicações com Lógica de Programação e Algoritmos

Este trabalho foi realizado na disciplina de Lógica de Programação e teve como foco o uso da linguagem Python para resolver problemas reais.

Foram desenvolvidas aplicações simulando sistemas de:

    📦 Loja de Vendas
    🍦 Sorveteria
    📚 Gerenciamento de Biblioteca
    🖨️ Sistema de uma loja de Copiadora

Todos os sistemas foram feitos com base em lógica sequencial, condicional e de repetição, com foco em boas práticas de código e estruturação de algoritmos.

Tecnologia utilizada: Python

🗃️ Projeto 2: Banco de Dados para Locadora de Veículos

Projeto desenvolvido na disciplina de Banco de Dados, com foco em modelagem relacional e criação de estruturas de dados no MySQL.

O sistema simula uma locadora de veículos, contendo:

    🔑 Criação de tabelas com chaves primárias e estrangeiras
    📊 Modelo relacional com integridade de dados
    🛠️ Implementação no MySQL Workbench

Esse projeto me proporcionou uma visão prática de como modelar e implementar um banco de dados relacional do zero.

Tecnologias utilizadas: MySQL, Workbench
